# 操作手册

## 📌 每日操作流程

### 早晨（9:30前）
1. 检查隔夜持仓
2. 查看大盘支撑压力位

### 盘中（9:30-15:00）
1. 监控b1/b2标的
2. 识别KDJ J值大负
3. 执行买入操作

### 收盘后（16:00）
1. 运行选股分析
2. 生成日报
3. 准备明日策略

## 🎯 b1识别流程

```
股价回调 → 检查支撑位 → 等待J值大负（J＜-5）
     ↓
是b1？ → 分批买入 → 设置止损
     ↓
否 → 继续等待
```

## 🎯 b2识别流程

```
b1反弹 → 出现中长阳（>3%）→ 确认突破
     ↓
是b2？ → 轻仓追涨 → 设置止损
     ↓
否 → 继续等待
```

## 📊 止损规则

### b1止损
```
买入K线最低价 - 2~5个价位
```

### b2止损
```
阳线最低价 - 2~5个价位
```

## 📈 仓位管理

- **70%主配置**：确定性高的标的
- **30%量化灵动**：b1/b2机会

## 🔍 检查清单

- [ ] baostock连接正常
- [ ] 股票数据获取成功
- [ ] MA计算正确
- [ ] 支撑压力位合理
- [ ] 预测逻辑自洽
- [ ] 建议操作明确

## 💡 常见问题

### Q: 没有b1信号怎么办？
A: 等待，不强求。空仓也是正确选择。

### Q: b2追涨被套怎么办？
A: 严格执行止损纪律，只输一根K线。

### Q: 如何确认J值？
A: 需要使用通达信/同花顺查看实时KDJ指标。
